Assumptions and Caveats

Noise-degraded DRI range is a heuristic: range_effective = range_nominal * min(1, sqrt(SNR / SNR_threshold)). It is intended for sensitivity exploration and discussion, not formal performance certification.
  • This is an exploratory engineering model, not a validated range prediction tool.
  • Johnson criteria are approximate and task-dependent.
  • NETD, FPN, and drift terms are simplified and treated as independent RMS contributors.
  • Range degradation is heuristic and not a validated Johnson/SNR probability model.
  • 1/f noise is modeled heuristically unless measured PSD data are available.
  • True 1/f impact requires measured noise PSD, correction strategy, bandwidth, and temporal/spatial processing assumptions.
  • Drift may appear as offset drift, gain drift, row/column drift, or spatially correlated clutter.
  • Atmospheric transmission, target contrast, optics, image processing, and algorithm performance can dominate real-world results.
  • This tool is intended to identify measurement requirements and viability regions, not certify performance.
  • Tracking jitter is approximated as 1 / SNR in pixels, for quick mission-utility screening only.
